Notifications

What Notifications Does TradingPlan Send?

TradingPlan can send you reminders to run your trading routines. That's it — there are no marketing notifications, no trade signals, and no spammy badges.

Notifications are entirely optional and must be explicitly enabled.

Setting Up Routine Reminders

Routine reminders notify you at a scheduled time to run your routine flow — useful for making pre-market preparation a consistent habit.

Enable Notifications

  1. Go to Profile → Notifications
  2. Toggle notifications on
  3. If prompted by iOS/macOS, grant TradingPlan permission to send notifications

Set a Reminder for a Specific Routine

  1. Go to Trade Plan → Routines
  2. Tap the routine you want a reminder for
  3. Open the routine Settings tab
  4. Enable a Schedule with your preferred days and time
  5. Set a Reminder Offset (e.g. 15 minutes before)

The notification will appear before the scheduled time on the days you've selected, with a tap-to-open shortcut that launches straight into the routine flow.

For example, a routine scheduled at 08:00 with a 15-minute reminder will notify you at 07:45 on each scheduled day.

Turning Off Notifications

From Within TradingPlan

  1. Go to Profile → Notifications
  2. Toggle notifications off

From System Settings (iPhone/iPad)

  1. Open Settings → Notifications → TradingPlan
  2. Toggle Allow Notifications off

From System Settings (Mac)

  1. Open System Settings → Notifications → TradingPlan
  2. Turn off notifications

Notifications Aren't Arriving

If you've set up notifications but they're not appearing:

  1. Check app notification permission: Settings → Notifications → TradingPlan — make sure "Allow Notifications" is on
  2. Check Do Not Focus is not blocking them: Settings → Focus → check whether your active Focus mode is filtering TradingPlan notifications
  3. Check the notification time is set correctly: In TradingPlan, go to Profile → Notifications and verify the scheduled time
  4. Check the routine is active: Notifications only fire for active routines (routines with the Active toggle enabled)

Notification Badge Count

TradingPlan does not use badge counts on the app icon.

The bell icon on the Dashboard shows a red dot when there are unread update notes (new features, release notes). Tap it to read the updates and the dot clears.

Spotlight and Siri Suggestions

TradingPlan indexes your strategies and routines for Spotlight search (iOS and macOS). You can search for your strategy by name from the home screen search or from macOS Spotlight, and tap the result to open it directly in the app.

Siri Shortcuts are not currently supported, but routine notifications include a direct link to open the flow immediately.
