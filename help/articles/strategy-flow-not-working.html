#108 Troubleshooting Last updated: April 2026

Strategy Flow Not Progressing

The Flow Won't Start

If the "Start Flow" button isn't appearing or does nothing:

  1. Make sure the strategy has at least one rule. An empty strategy has nothing to check — the flow needs content to display
  2. Make sure the strategy is saved — if you just created it, tap Save before trying to start a flow
  3. Try navigating away (go to the Dashboard, then back to Strategies) and starting the flow again

The Flow Is Stuck on a Section

If the flow appears frozen or won't advance past a section:

  1. Check that you've responded to all required items in the current section — some items require a selection or confirmation before the flow can advance
  2. Scroll down within the current section — there may be more items below the visible area that need attention before you can proceed
  3. Force-quit the app and reopen it, then return to the flow — your progress should be preserved

Tapping Doesn't Register

On touchscreen devices, if taps aren't registering:

  1. Try tapping more firmly or slowly — very fast taps can sometimes miss small tap targets
  2. Check if your screen protector is affecting touch sensitivity
  3. Try using the app with a different tap — sometimes a swipe-up to full-screen or a two-finger tap can interrupt expected behaviour

On Mac, make sure you're clicking (not just hovering) on interactive items.

The Flow Restarted From the Beginning

The flow resets if:

  • You edited the strategy mid-flow (the app warns you this will happen)
  • The app was force-quit in a way that didn't preserve state
  • You tapped the reset option

If you didn't intend to reset it, simply start the flow again from the beginning.

The Flow Is Running But My Strategy Has Different Rules Now

If you edited your strategy rules and the flow is showing old rules:

  1. Exit the flow
  2. Confirm your strategy edits are saved (tap Save in the strategy editor)
  3. Start a new flow — it will use the current, saved version of the rules

Completed a Flow But It Didn't Appear in Flow History

Flow History records completed flows. A flow only registers as complete when you reach and confirm the final completion screen. If you closed the app before reaching that screen, it won't be recorded.

To confirm it was recorded:

  1. Go to Trade Plan → Flow History
  2. Check the date/time of the most recent entry

Flow Keeps Asking for the Same Strategy

If you're accidentally in the wrong strategy's flow:

  1. Exit the current flow
  2. Go to the Strategies tab
  3. Tap the specific strategy you want to trade
  4. Start the flow from there

Still Having Problems?

If none of the above resolves it, please report the issue via Profile → Help & Feedback, describing:

  • Which strategy (how many rules, which sections have content)
  • What exactly happens when you try to advance
  • Your device and iOS version

This helps us identify any edge cases in the flow logic.
